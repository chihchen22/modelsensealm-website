Iconography
Icons

All icons are hand-drawn SVG using stroke-linecap="square" and stroke-linejoin="miter" at stroke-width="1.5". The geometric, hard-cornered style reinforces rigor and precision. Never use rounded corners or filled icon variants.

Arrow Right
Arrow Diagonal
Chart Line
Chart Bar
Hamburger

Icon language: square linecaps, miter joins, stroke-width 1.5. Consistent geometric register across all marks.

dash-card · all .4s cubic-bezier(.16, 1, .3, 1)
service-bar · width .7s ease
scroll reveal · all .8s cubic-bezier(.5, 0, 0, 1) · Y +30px → 0
arrow translate · transform .3s

Easing philosophy: standard interactions use ease-out. High-impact card flips use cubic-bezier(.16, 1, .3, 1) — a fast start, smooth deceleration. Scroll reveals use cubic-bezier(.5, 0, 0, 1) for a weighty, deliberate entrance.

Brand Voice
Tone & Messaging
Voice Principles
Direct over diplomatic
Surface problems explicitly. Challenge weak premises. The audience is a senior practitioner, not a student.
Transparent over opaque
Make assumptions explicit, uncertainty quantifiable, sensitivity testable. No black-box framing.
Rigorous, not academic
Bridge the gap between theory and institutional reality. Models that practitioners can actually build.
Intelligence with judgment
Models are math. Risk is human. Quantitative precision serves human decision-making, not the reverse.
